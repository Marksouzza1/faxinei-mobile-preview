🧹 Faxinei

Faxinei é um aplicativo mobile idealizado para conectar diaristas e contratantes de forma simples, rápida e segura.

O projeto surgiu como Trabalho de Conclusão de Curso (TCC) no curso de Análise e Desenvolvimento de Sistemas da Faculdade Senac Jaraguá do Sul, com o objetivo de resolver dificuldades reais no processo de contratação de serviços domésticos.

📱 Sobre o projeto

No Brasil, milhões de profissionais atuam no setor de serviços domésticos, mas a conexão entre diaristas e contratantes ainda acontece, em grande parte, de forma informal — através de indicações ou classificados.

O Faxinei propõe uma solução digital para esse problema, funcionando como um marketplace mobile onde:

Diaristas podem divulgar seus serviços
Contratantes podem encontrar profissionais confiáveis
A comunicação ocorre de forma direta e organizada
O processo de contratação se torna mais seguro e transparente
🚀 Objetivo

Desenvolver uma aplicação que:

Facilite a conexão entre diaristas e contratantes
Aumente a visibilidade dos profissionais
Ofereça uma experiência intuitiva e acessível
Promova segurança e confiança entre os usuários

Cadastro e login (diarista ou contratante)
Criação e edição de perfil
🧹 Diaristas
Cadastro de serviços
Definição de preço e disponibilidade
Exibição de perfil profissional
🔍 Contratantes
Busca de diaristas por filtros:
localização
preço
avaliações

Agendamento de serviços (planejado)

🧠 Arquitetura (visão inicial)

O projeto foi pensado utilizando uma abordagem mobile-first com backend serverless:

Aplicação Flutter (Android / iOS / Web)
Firebase como BaaS
Estrutura modular por features
📊 Metodologia

O desenvolvimento seguiu etapas de engenharia de software:

Levantamento de requisitos com usuários reais
Modelagem do sistema
Definição de requisitos funcionais e não funcionais
Prototipação e planejamento técnico
Abordagem incremental (MVP)
